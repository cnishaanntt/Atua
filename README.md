Atua
===================================================
The keeper


### What is it?


A  Progressive Web application that consists of a collection of lambda functions. It utilizes Amazon S3, Amazon API Gateway to analyze uploaded images in S3 with Amazon Rekognition OCR Capabilities. It follows HL7's FHIR for data inter-operability procedures.
